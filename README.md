# Spotibot

[https://spotibot-psi.vercel.app/](https://spotibot-sigma.vercel.app/)

## 🌟 基本機能 (Standard Mode)

### 🎵 Spotify Music Sync
- 再生中の楽曲のアルバムアートから主要色を抽出し、照明をシンクロさせます。
- 楽曲のテンポやエナジーに合わせて、照明の明るさや色が動的に変化します。
- 現在再生中のトラック情報、BPM、キー、ムードなどの音楽的特徴を表示します。

### 🏠 Device Control
- 全デバイスの一括電源ON/OFF。
- 明るさ（0-100%）の無段階調整。
- RGBカラーピッカーによる自由な色設定。
- 色温度（ホワイトバランス）の調整（2700K - 6500K）。
- プリセット保存機能で、お気に入りのシーンをワンタップで呼び出し可能。

---

## 🚀 高度な機能 (Live Venue Mode)

「Multi-Color Sync (Live Venue Mode)」をオンにすることで、より没入感のある照明体験が可能になります。

### 🌈 Multi-Color Immersion
- アルバムアートから複数の色（パレット）を抽出し、複数のデバイスに異なる割り当てます。
- 単一色ではなく、空間全体でアルバムの世界観を表現します。

### 🖱️ Drag & Drop Color Routing
- **直感的な操作**: 画面上のデバイスチップをドラッグし、アルバムアート横の「カラーポインター」やコントロールパネル内の「パレットドット」にドロップすることで、特定の色をそのデバイスに割り当てることができます。
- **Auto設定**: ルーティングを解除すると、自動的に最適な色が割り当てられます。

### 💡 Light Grouping
- **グループ作成**: 複数の照明をまとめて「リビング」「寝室」などのグループを作成できます。
- **一括操作**: グループ単位で選択し、色や明るさを変更できます。

### 🎛️ Individual Control
- **個別制御モード**: 特定の照明だけを別の色に変えたい場合、「Individual Ctrl」にチェックを入れることで、個々のデバイスごとの色設定が可能になります。
- 選択したデバイスの横に専用のカラーピッカーが表示されます。
